TinyToken SDK

Python SDK for Text Compression API

Installation

Install the SDK using pip:

pip install tinytoken-sdk

Requirements

  • Python 3.7 or higher
  • requests >= 2.20.0

Quick Start

Here's a simple example of how to use the SDK:

import tinytoken

# Simple usage
result = tinytoken.compress("Your text here")
print(result)

# Using the client class
client = tinytoken.TinyToken()
result = client.compress("Your text here")
print(result)

Response Format

The compress method returns a dictionary containing:

Field Type Description
original_text string The input text
shortened_text string The compressed version
original_tokens integer Number of tokens in original text
shortened_tokens integer Number of tokens in compressed text
reduction_percent float Percentage of token reduction
similarity_score float Semantic similarity between original and compressed text
